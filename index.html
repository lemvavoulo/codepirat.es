The Code Pirates

The Code Pirates is a handpicked selection of popular Open Source Github projects, that are hosted on a Content Delivery Network (CDN). The content we deliver is not hosted on Github.

The reasons behind The Code Pirates are as follows:

  • To allow others to remotely link, or offload their assets, similar to projects like BootstrapCDN, CDNJS, jsDeliver, and CDN.TL
  • To introduce those who are new to Github to some great open source projects.
  • To offer a way to access repos should Github's servers go down.
    This is not the main reason behind the project, however.

★ Notes ★

Whilst the main purpose of this project is to serve assets from a CDN, we understand that Github's servers can go down without notice, and this is a temporary, albeit limited solution, to that problem. Our service is not 100% reliable, and we make no claims to that.

Since this project is hosted on Github Pages, it is liable to go offline. We have attached the domain to Cloudflare, so our page should persist even if Github Pages does go offline.
